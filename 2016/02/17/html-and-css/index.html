html_and_css

Html+CSS:

  • HTML是网页内容的载体。内容就是网页制作者放在页面上想要让用户浏览的信息，可以包含文字、图片、视频等。
  • CSS样式是表现。就像网页的外衣。比如，标题字体、颜色变化，或为标题加入背景图片、边框等。所有这些用来改变内容外观的东西称之为表现。
  • JavaScript是用来实现网页上的特效效果。如：鼠标滑过弹出下拉菜单。或鼠标滑过表格的背景颜色改变。还有焦点新闻（新闻图片）的轮换。可以这么理解，有动画的，有交互的一般都是用JavaScript来实现的。

常用标签

  • p 段落
  • hx 不同的标题大小
  • em 斜体
  • stong 粗体
  • span 自定义样式（配合css使用）
  • q 引用（会自动添加上“”，但觉得并没有什么屌用）
  • br 换行（相当于回车）
  • address 地址（说白了就是把字体默认为斜体，可以通过css修改，发现也并没有 屌用，感觉用span可以替代）
  • code 插入一句代码（会使排版有些变化）
  • pre 插入一段代码
  • ul（ol） li 组成列表
  • div 分块
  • table 表格 tr 表示行 td th表示列，th多用于表示表的第一列
  • caption 表格标题
  • a 链接 herf链接地址
  • img 图片 src 图片url

表单

type类型

  • text 普通输入文本框
  • passwod 密码输入框
  • submit 提交按钮
  • reset 充值按钮
  • checkbox 复选框
  • radio 单选框

< textarea rows=”行数” cols=”列数”> 大段文本输入框
< select> 下拉列表
< option value=”向后台的提交值”>展示值</ option>
< label for=”与之配对的input的name”> 普通文本
